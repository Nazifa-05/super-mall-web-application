#  Super Mall Web Application

A full-stack Mall Web Application built using **React, Tailwind CSS, Node.js, Express, and MongoDB**.

This project allows users to browse mall shops, explore categories, compare products, and register/login securely using JWT authentication.

---

## 🚀 Features

### 🔹 User Side
- 🏠 Modern Home Page (Red & White Theme)
- 🗂️ Shop by Categories
- 🏬 Shop Listing Page
- 🛍️ Dynamic Shop Details Page
- 🔥 Offers Page
- ⚖️ Compare Products Page
- 🔐 User Authentication (Register & Login)
- 🧾 JWT Based Authentication
- 📱 Fully Responsive Design

### 🔹 Backend
- 🗄️ MongoDB Database
- 🔑 JWT Authentication
- 🔒 Password Hashing using bcrypt
- 🌐 REST API using Express
